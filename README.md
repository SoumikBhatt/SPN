# SPN
SPN is a food ordering app for restaurants based in London. It is a portable application that can be used in multiple restaurants.


## Tools & Technology Used

- Flutter
- Dart
- Flutter BLoc
- http
- Stripe Payment SDK
- SQFLite
- Background Fetch
- Firebase
- Push Notifications
- Material Design
